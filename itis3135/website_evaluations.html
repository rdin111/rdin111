Website Evaluations

Good Website: Airbnb

Description: Airbnb is a globally recognized platform that allows users to book accommodations and experiences. It features a clean and modern design that prioritizes usability and aesthetics.

  • Contrast: Uses a clean layout with high contrast elements that make navigation intuitive.
  • Repetition: Consistent fonts, colors, and interface elements create a seamless user experience.
  • Alignment: Well-structured grid layout ensures content is easy to read and navigate.
  • Proximity: Related elements are grouped together, making information easy to find.

Bad Website: Gates N Fences

Description: Gates N Fences is an online retailer for driveway gates, but its website is outdated, cluttered, and difficult to navigate.

  • Contrast: Poor use of background textures and text colors make readability difficult.
  • Repetition: Inconsistent fonts and button styles create a disjointed experience.
  • Alignment: Elements are scattered, and text alignment is inconsistent.
  • Proximity: Information is not well-grouped, leading to a cluttered and confusing layout.

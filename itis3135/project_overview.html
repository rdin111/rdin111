Project Overview

CodeStarter: Learn Python

Project Summary

CodeStarter is a simple, beginner-friendly site to help people learn the basics of Python. It’ll have short lessons, a small quiz, and a contact form — perfect for anyone just getting started with programming.

I'm building this site for a client/friend, Tim Burns, who wanted something straightforward for new learners. The site will have five pages: Home, About, Tutorials, Quiz, and Contact.

Client Info

  • Name: Tim Burns
  • Connection: External Client / Friend
  • Email: burnstim86@gmail.com

Who It's For

This site is for people who are just starting out with Python:

  • Students learning to code
  • Self-taught beginners
  • Anyone prepping for coding bootcamps or interviews

What's On the Site

  • Quick, easy Python lessons (with examples)
  • A short multiple-choice quiz
  • Info about the site and creator
  • A form for feedback or questions

Wireframe

Site Map

Pages

1. Home

  • Quick intro to the site
  • Button that takes users to the tutorials

2. About

  • Why the site was made
  • Short bio (with collapsible section)

3. Tutorials

  • Lessons on Python topics like variables, loops, and functions
  • One topic per tab for easy navigation

4. Quiz

  • 5 multiple-choice questions
  • Instant feedback using JavaScript

5. Contact

  • Form with name, email, and message fields
  • Shows a thank-you message after submitting

Interactive Features

About Page – Accordion

  • Click to expand/collapse the creator bio

Tutorials Page – Tabs

  • Each Python topic is in its own tab

Quiz Page – Instant Scoring

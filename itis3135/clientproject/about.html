About CodeStarter

CodeStarter is a project designed to help beginners start their Python programming journey in a simple and supportive way. Whether you're a student, a career switcher, or just curious about coding, this site gives you the basics to get started.

Who Built This?

Hi! I'm Rahul Dinesh, a student at UNC Charlotte. I created this website as part of my web development course to make learning Python easier for others.

My client and friend, Tim Burns, provided the idea and support for building this site. Together, we shaped CodeStarter to be beginner-friendly, straightforward, and fun to use.

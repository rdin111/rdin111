  • Lesson 1: Printing and Variables

    The first step in many programming languages is learning how to display output. In Python, we use the print() function.

    print("Hello, World!")

    Variables are used to store data. Assign values using =.

    message = "Welcome to CodeStarter!"

  • Lesson 2: Conditional Statements

    Conditional statements allow code to make decisions using if.

    temperature = 30
if temperature > 25:
    print("It's a warm day!")

    Use else and elif for additional conditions.

    score = 75
if score >= 60:
    print("You passed!")
else:
    print("You need to study more.")
    grade = 85
if grade >= 90:
    print("Grade: A")
elif grade >= 80:
    print("Grade: B")
elif grade >= 70:
    print("Grade: C")
else:
    print("Grade: F")

  • Lesson 3: Loops

    Loops repeat blocks of code. Use for or while.

    for i in range(5):
    print(f"Number: {i}")

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(f"I like {fruit}s")
    count = 0
while count < 3:
    print(f"Count is {count}")
    count += 1

  • Lesson 5: Lists and Dictionaries

    Lists store ordered items. Dictionaries store key-value pairs.

    my_list = ["red", "green", "blue"]
print(my_list[0])
my_list.append("yellow")
print(my_list)
    student = {"name": "Charlie", "major": "Computer Science", "gpa": 3.8}
print(student["name"])
print(student.get("major"))

student["year"] = 3
